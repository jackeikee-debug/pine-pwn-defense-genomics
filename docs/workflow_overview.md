# Workflow overview

The workflow has eight connected evidence layers: metadata and proteome quality control, longest-isoform selection, orthology, domain and defense-module annotation, nematode secretome prioritization, host-module mapping, gene-family diagnostics, and infection-expression/network/structure integration.

Candidate orthogroups provide the common coordinate system across pine species. Nematode proteins are screened for secretion and functional evidence, then linked to biologically plausible host modules through annotation, homology, compartment compatibility, network position, expression response, and selected structure quality. Each layer remains traceable to its source table and generating command.

The final rankings are hypothesis-generating. Predicted links, interologs, functional associations, and cross-species expression concordance are not direct physical-interaction evidence.
